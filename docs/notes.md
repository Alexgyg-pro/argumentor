##### Log file

Les logs sur la console c'est pratique mais ça devient vite encombrant. Ce serait bien de réserver les logs sur la console pour la recherche d'un bug puis le supprimer une fois que c'est réglé. En revanche, ce serait bien aussi d'avoir un fichier journal qui répertorie l'activité.
